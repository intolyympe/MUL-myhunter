# MUL-my_hunter

Epitech project — Duck Hunt legacy game in C with CSFML.

## Usage

```bash
make
./my_hunter
```

## Description

`./my_hunter`

The player is a hunter who must shoot ducks before they escape:
- Ducks appear on screen and move from one side to the other.
- Click on a duck to shoot it.
- Missing too many ducks costs a life.

## Controls

| Input | Action |
|-------|--------|
| Left click | Shoot a duck |
| ESC | Quit |

## Features

- Animated duck sprites with sprite sheets
- Score display
- Menu & Game Over screen
- Frame-rate independent animations with sfClock
- 1926x1080 window
